# 📱 Android Forensics Toolkit

## 🔍 Overview
Android Forensics Toolkit is a Python-based educational project designed for **authorized digital forensic analysis** using ADB.  
It helps in extracting, analyzing, and generating reports from Android devices.

---

## ⚙️ Features
- 📡 Device detection using ADB
- 📂 Data extraction (DCIM, Downloads, Documents)
- 🧠 Automatic file analysis (Images, Videos, Documents)
- 📊 Forensic report generation (TXT format)

---

## 🧰 Technologies Used
- Python 3
- Android Debug Bridge (ADB)
- OS Module

---

## 🚀 How to Run

```bash
pip install -r requirements.txt
python main.py
